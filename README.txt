README

This is the Clicky Analytics plugin for MyBB.

Plugin to insert the Clicky tracking code to pages.

Please, refer to the [INSTALL] file for installation instructions.


==== Description ====

This is very simple plugin to insert the Clicky tracking code to your pages.

It requires only to specify your Site ID.

==== Compatibility ==

The plugin is tested only with MyBB 1.6.x version.

Not tested with 1.4.x at all. Although, it may work.
